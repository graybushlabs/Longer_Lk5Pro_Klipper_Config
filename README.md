# LK5 Pro Configuration for Klipper
This is my custom configuration for Klipper. It is based on a Longer LK5 Pro that has been modified. Maybe you will find it useful too.

## Modifications
- Dual z screws
- Orbiter v1.5 extruder
- Klipper (Raspi 3b+ host)
- Webcam
- 3dTouch

Read more about it on my [blog](https://graybush.io/blog/longer-lk5-pro-mods/).
